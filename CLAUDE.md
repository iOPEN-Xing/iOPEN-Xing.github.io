# 项目说明

这是邢介政的个人简历站点，基于 academicpages / Minimal Mistakes Jekyll 模板整理。

## 编辑原则

- 简历事实以 PDF 原文为准。
- PDF 未出现的信息一律保留 `***` 或留空，不补写、不推测。
- 个人头像、校徽、组织 logo 来自 `D:\jianli\资料`。
- 页面排版可以优化，但不能改变已核验事实含义。

## 主要页面

- `_pages/about.md`：主页
- `_pages/experience.md`：经历页
- `_pages/projects.md`：项目页
- `_pages/cv.md`：背景页

## 本地预览

```bash
bundle install
bundle exec jekyll serve
```

修改 `_config.yml` 后需要重启 `jekyll serve`。
